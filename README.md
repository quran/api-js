A library for fetching quran data from the [quran.com API][qdc-api]. This library also works on both Node.js and the browser.

[Checkout Docs][docs]

## Installation

using npm:

```ssh
npm install @quranjs/api
```

using yarn:

```ssh
yarn add @quranjs/api
```

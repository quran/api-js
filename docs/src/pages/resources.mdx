## findAllChapterInfos

- [Quran.com API Docs](https://quran.api-docs.io/v4/resources/chapter-info)

**Params:**

`resources.findAllChapterInfos(options?: GetResourceOptions)`

| Param   | Type                 |
| ------- | -------------------- |
| options | `GetResourceOptions` |

**Example:**


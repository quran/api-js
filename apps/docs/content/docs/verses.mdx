---
title: Verses API
description: Retrieve Quran verses (ayahs) with translations, tafsirs, audio, and word-by-word breakdowns.
---

The Verses API provides comprehensive access to Quranic verses with support for translations, tafsirs (commentaries), audio recitations, and detailed word analysis.

## Methods Overview

```ts
client.verses.findByKey(key)        // Get verse by key (e.g., "2:255")
client.verses.findByChapter(id)     // Get all verses in a chapter
client.verses.findByPage(page)      // Get verses on a specific page
client.verses.findByJuz(juz)        // Get verses in a juz
client.verses.findByHizb(hizb)      // Get verses in a hizb
client.verses.findByRub(rub)        // Get verses in a rub
client.verses.findRandom()          // Get a random verse
```

## Get Verse by Key

Retrieve a specific verse using its unique key in "chapter:verse" format:

```ts
// Ayah al-Kursi (Verse of the Throne)
const ayahAlKursi = await client.verses.findByKey("2:255");

// First verse of Al-Fatiha
const firstVerse = await client.verses.findByKey("1:1");

// Last verse of the Quran
const lastVerse = await client.verses.findByKey("114:6");
```

### With Translations

```ts
const verse = await client.verses.findByKey("2:255", {
  translations: [20, 131], // Translation IDs
  words: true,             // Include word-by-word breakdown
  translationFields: {     // Control translation fields
    text: true,
    languageName: true,
    resourceName: true
  }
});
```

### With Audio

```ts
const verse = await client.verses.findByKey("1:1", {
  reciter: 2,  // Reciter ID for audio
  words: true  // Include word audio segments
});
```

## Get Verses by Chapter

Retrieve all verses from a specific chapter:

```ts
// Get all verses from Al-Fatiha
const fatihaVerses = await client.verses.findByChapter("1");

// Get verses with translations and pagination
const verses = await client.verses.findByChapter("2", {
  translations: [20],  // English translation
  perPage: 10,        // Limit to 10 verses
  page: 1,            // First page
  words: true         // Include word breakdown
});
```

## Get Verses by Page

Access verses that appear on a specific Mushaf page:

```ts
// First page of the Quran
const firstPageVerses = await client.verses.findByPage("1");

// With translations
const pageVerses = await client.verses.findByPage("42", {
  translations: [131], // Urdu translation
  words: false
});
```

Retrieve a random verse for daily inspiration:

```ts
const randomVerse = await client.verses.findRandom({
```

## Verse Data Structure

```ts
interface Verse {
  id: number;                    // Unique verse ID
  verseNumber: number;          // Verse number within chapter
  verseKey: string;             // "chapter:verse" format
  hizbNumber: number;           // Hizb number
  rubNumber: number;            // Rub number
  juzNumber: number;            // Juz number
  pageNumber: number;           // Mushaf page number
  textUthmani: string;          // Arabic text (Uthmani script)
  textIndopak: string;          // Arabic text (Indopak script)
  textImlaei: string;           // Arabic text (Imlaei script)
  words?: Word[];               // Word-by-word breakdown
  translations?: Translation[]; // Verse translations
  tafsirs?: Tafsir[];          // Verse commentaries
  audio?: AudioResponse;        // Audio recitation data
}
```

## Advanced Options

### Word Fields

Control which word data to include:

```ts
const verse = await client.verses.findByKey("1:1", {
  words: true,
  wordFields: {
    text: true,           // Arabic text
    transliteration: true, // Romanized text
    translation: true,     // Word translation
    audio: true           // Word audio
  }
});
```

### Translation Fields

Control translation data included:

```ts
const verse = await client.verses.findByKey("2:255", {
  translations: [20, 131],
  translationFields: {
    text: true,          // Translation text
    languageName: true,  // Language name
    resourceName: true,  // Translator name
    resourceId: true     // Translation resource ID
  }
});
```

### Verse Fields

Control main verse data:

```ts
const verse = await client.verses.findByKey("1:1", {
  fields: {
    textUthmani: true,    // Arabic Uthmani script
    textIndopak: false,   // Skip Indopak script
    textImlaei: false,    // Skip Imlaei script
    chapterName: true,    // Include chapter name
    pageNumber: true,     // Include page number
    juzNumber: true       // Include juz number
  }
});
```

## Pagination

For methods that return multiple verses:

```ts
const verses = await client.verses.findByChapter("2", {
  page: 1,        // Page number (1-based)
  perPage: 20,    // Verses per page
  translations: [20]
});

// Access pagination info from response metadata
console.log(`Page ${verses.pagination?.currentPage} of ${verses.pagination?.totalPages}`);
```

## Error Handling

The API validates inputs and provides clear error messages:

```ts
try {
  // Invalid verse key format
  await client.verses.findByKey("invalid");
} catch (error) {
  console.error(error.message); // "Invalid verse key"
}

try {
  // Non-existent verse
  await client.verses.findByKey("1:10"); // Al-Fatiha only has 7 verses
} catch (error) {
  console.error(error.message); // API error response
}
```

## Common Use Cases

### Reading Interface

```ts
async function getVerseForReading(verseKey: string) {
  const verse = await client.verses.findByKey(verseKey, {
    translations: [20, 131], // English and Urdu
    words: true,
    wordFields: {
      text: true,
      transliteration: true,
      translation: true
    }
  });

  return {
    arabic: verse.textUthmani,
    translations: verse.translations,
    wordByWord: verse.words,
    location: {
      chapter: verseKey.split(':')[0],
      verse: verse.verseNumber,
      page: verse.pageNumber,
      juz: verse.juzNumber
    }
  };
}
```

### Study Interface

```ts
async function getVerseForStudy(verseKey: string) {
  const verse = await client.verses.findByKey(verseKey, {
    translations: [20], // English
    tafsirs: [171],     // Tafsir ibn Kathir
    words: true,
    wordFields: {
      text: true,
      transliteration: true,
      translation: true,
      audio: true
    }
  });

  return {
    verse,
    commentary: verse.tafsirs?.[0]?.text,
    wordAnalysis: verse.words
  };
}
```

### Chapter Navigation

```ts
async function getChapterWithNavigation(chapterId: string, currentPage = 1) {
  const verses = await client.verses.findByChapter(chapterId, {
    page: currentPage,
    perPage: 10,
    translations: [20],
    words: false
  });

  return {
    verses,
    hasNext: currentPage < (verses.pagination?.totalPages || 1),
    hasPrev: currentPage > 1,
    totalPages: verses.pagination?.totalPages
  };
}
```

## Parameters Reference

### GetVerseOptions

```ts
interface GetVerseOptions extends BaseApiParams, PaginationParams {
  reciter?: string | number;        // Reciter ID for audio
  words?: boolean;                  // Include word breakdown
  translations?: (string | number)[]; // Translation resource IDs
  tafsirs?: (string | number)[];    // Tafsir resource IDs
  wordFields?: Partial<Record<WordField, boolean>>;
  translationFields?: Partial<Record<TranslationField, boolean>>;
  fields?: Partial<Record<VerseField, boolean>>;
}
```

### PaginationParams

```ts
interface PaginationParams {
  page?: number;     // Page number (1-based)
  perPage?: number;  // Items per page
}
```
